# Blind Vision — AI-Powered Assistive Navigation System

Blind Vision is an assistive technology project designed to help visually impaired individuals navigate their surroundings more safely and independently. The system combines embedded hardware, computer vision, and audio feedback to provide real-time environmental awareness.

## Features

* Obstacle detection using ultrasonic sensors
* Image recognition using OpenCV
* Text detection and reading using Tesseract OCR
* Real-time audio feedback for users
* Portable Raspberry Pi-based architecture
* Modular design for future object detection integration

## Technologies Used

### Software

* Python
* OpenCV
* Tesseract OCR
* dlib

### Hardware

* Raspberry Pi
* Camera Module
* Ultrasonic Sensors
* Audio Output/Buzzer
* Portable Battery System

## System Overview

The system uses a Raspberry Pi as the central controller. Camera input is processed using computer vision techniques to identify visual information and detect text within the environment. Ultrasonic sensors measure distances to nearby obstacles, while audio feedback informs the user about detected objects and hazards.

## Project Goals

* Assist visually impaired users with navigation
* Improve environmental awareness through computer vision
* Detect and communicate textual information through OCR
* Provide low-cost and portable accessibility technology
* Explore object detection techniques for future development

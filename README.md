The Blind Assistant System is an embedded solution designed to assist visually impaired individuals with safe navigation. 


The system uses sensors and real-time processing to detect obstacles in the environment and 


provide feedback to the user, improving mobility and independence.


Objectives


Assist visually impaired users in navigating their surroundings


Detect nearby obstacles in real time


Provide immediate feedback to enhance safety


Build a low-cost, portable assistive device


System Architecture

The system is built around a Raspberry Pi, which processes sensor data and triggers feedback mechanisms.


Main Components

Raspberry Pi (central controller)

Ultrasonic sensors (distance measurement)

Camera module 

Audio output / buzzer 


Power supply (battery-based system)
